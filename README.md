# Pakistani Ramadan 30-Day Menu Planner

**Ramadan Mubarak! 🌙**  
A simple, practical 30-day Sehri & Iftar menu specially designed for Pakistani households — featuring all the classic desi favorites that make Ramadan feel like home.

This planner helps take the daily "what to cook?" stress away so you can focus on fasting, prayers, Quran, and family time.

### Features

- **30 full days** of balanced Sehri (Suhoor) and Iftar ideas
- Authentic Pakistani dishes: pakoras, samosas, anda paratha, dahi bhallay, nihari, haleem, chicken karahi, biryani, sheer khurma, fruit chaat, lassi, rooh afza, and more
- Emphasis on sustaining foods for long fasting hours (protein + complex carbs + hydration)
- Practical rotation of favorites — realistic for most Karachi/Lahore/Punjabi/Sindhi kitchens
- Prep-ahead friendly (freeze kebabs, shami, samosas in bulk)
- Tips included: moderation, avoiding excess salt/spice, family scaling

### What's Inside the PDF

- Day-by-day breakdown (Day 1 to Day 30)
- Sehri: Quick, filling options (parathas, daal, anda, chana, oats, etc.)
- Iftar: Classic sequence → Dates + sharbat → Fried snacks → Chaat/soup → Main curry/rice → Dessert
- Compact layout — easy to print (A4) or view on phone

### How to Use

1. Download the PDF from the **Releases** section (or view it directly in the repo if you prefer)
2. Print it or save to your gallery/notes app
3. Prep in advance — many items freeze beautifully
4. Adjust spice levels, portion sizes, or swap chicken/mutton/beef as needed
